# 🎬 Gait Analyzer - Pose Video Player

## 🚀 JETZT STARTEN

### Methode 1: Direkt öffnen (EMPFOHLEN!)
1. **Doppelklick** auf diese Datei:
   ```
   C:\Users\Nutzer\.openclaw\workspace\app\pose-player.html
   ```
2. **Video wählen** (aus `data/raw/`)
3. **JSON wählen** (aus `data/processed/`)
4. **"▶️ Player starten"** klicken
5. **Abspielen!** 🎉

### Methode 2: Über Streamlit
1. Streamlit Dashboard öffnen (F5)
2. Auf **"▶️ POSE PLAYER ÖFFNEN"** klicken

## ✨ Features

- **🎥 EIN Video mit Pose-Overlay** - nicht getrennt!
- **▶️ Echtes Playback** - Video läuft mit synchroner Pose
- **⏯️ Vollständige Controls** - Play, Pause, Seek, Speed
- **🎨 Anpassbar** - Confidence, Skeleton, Points ein/aus
- **⚡ Smooth** - Kein Ruckeln, kein Flackern

---

## 🎮 Steuerung

| Button | Funktion |
|--------|----------|
| ⏮️ Start | Zum Anfang |
| ⏮️ -10s / -1s | Zurück spulen |
| ▶️ / ⏸️ | Play / Pause |
| ⏭️ +1s / +10s | Vor spulen |
| 🐌 / 🚀 | Slower / Faster |
| Progress Bar | Direkt springen |

- **Skeleton** - Knochenverbindungen ein/aus
- **Points** - Keypoints ein/aus
